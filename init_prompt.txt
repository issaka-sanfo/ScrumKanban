You are a senior full-stack software architect and developer.

Your task is to generate a complete production-ready Agile Scrum Kanban Board application.

The goal of the application is to help teams manage Scrum projects using a Kanban board with the following features:

MAIN FEATURES
- Create and manage Scrum Projects
- Manage Product Backlog
- Create Sprints
- Drag & Drop Kanban Board
- Task lifecycle (To Do, In Progress, Code Review, Done)
- Assign tasks to users
- Story points
- Labels and priorities
- Comments on tasks
- Sprint progress tracking
- Authentication (JWT)
- Role management (ADMIN, SCRUM_MASTER, DEVELOPER)
- Dashboard with sprint metrics
- REST API documentation

TECHNOLOGY STACK

Architecture
- Clean Architecture using Ports and Adapters (Ports / Services)
- Domain Driven Design principles
- Separation of:
  - Domain
  - Application
  - Infrastructure
  - Web

Backend
- Java 21
- Spring Boot
- Spring Data JPA
- Spring Security + JWT
- MapStruct
- Flyway for database migrations
- Lombok
- OpenAPI (Swagger)

Database
- PostgreSQL

Frontend
- Angular (latest version)
- Angular Material
- RxJS
- NgRx for state management
- Drag and Drop using Angular CDK

PROJECT STRUCTURE
